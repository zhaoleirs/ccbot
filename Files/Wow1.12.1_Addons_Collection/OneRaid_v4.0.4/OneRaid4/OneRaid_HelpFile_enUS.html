Slash Commands


|cFFFF0000* Requires the user to be an assistant or leader.


|cFFFF0000** Requires the user to be an raid leader.


/ornotes

|cFFFFFFFF - Opens the version notes window.


/orpassword

|cFFFFFFFF - Sets the password for certain whisper commands.


/orcure

|cFFFFFFFF - Attempts to cure a debuff.


/orassist

|cFFFFFFFF - Assists the main assist


/ors message *

|cFFFFFFFF - Sends message to the raid and displays it on screen.

|cFF00FF00 > Example: /ors Boss incoming in 10 seconds


/orsound soundfile *

|cFFFFFFFF - Plays soundfile for everyone in raid to hear.

|cFF00FF00 > Example: /orsound Sound\\Doodad\\BoatDockedWarning.wav


/ording [message] *

|cFFFFFFFF - Sends a ding to the raid, if message is provided, it is displayed on screen.

|cFF00FF00 > Examples: /ording INCOMING, /ording


/orres *

|cFFFFFFFF - Displays the resists of everyone in raid.


/ordur *

|cFFFFFFFF - Displays the durability of everyone in raid.


/oritem itemname

|cFFFFFFFF - Displays how many itemname everyone has in their inventory.

|cFF00FF00 > Examples: /oritem [Hearthstone], /oritem Heavy Runecloth Bandage


/ordisband *

|cFFFFFFFF - Disbands the entire raid.


/orreform *

|cFFFFFFFF - Disbands the raid and reforms it with the same members.


/orready *

|cFFFFFFFF - Performs a ready check, anyone not responding to the popup within 10 seconds is marked AFK.


/orswitch [player1] [player2]/[groupNumber] *

|cFFFFFFFF - Switchs a players raid group without needing to be leader. You can use "player" or "target" in place of a players name to move either yourself or your current target. If player1 or player2/group are not provided, then you will switch yourself with your current target, provided your target is a raid member. If only a group number is provided you will moved yourself to that group. If only player1 is provided, they will be switched with yourself. If you try to move someone to a full group, they will be switched with someone that is either dead, a ghost, or offline.. provided there is someone that meets those requirements.

|cFF00FF00 > Examples: /orswitch Player1 Player2, /orswitch Player1, /orswitch 3, /orswitch player target


/orsave **

|cFFFFFFFF - Save every member of the raids current group. If a new member joins the saved positions will be lost.


/orload **

|cFFFFFFFF - Resets the raid members groups to what it was when /orsave was used.


/orinvite [level] [z-"[zone]"] [r-"rank"] *

|cFFFFFFFF - Invites members of your guild to raid.

|cFF00FF00 > Examples: /orinvite 60 z-"", /orinvite 55-60 z-"Blackrock Deeps" r-"Alt"


|cFFFFFFFF - If level parameter is not provided, it will invite everyone between level 1 and 60. Level can be a ranged value (55-60) or a single level (60).

|cFF00FF00 > Examples: 55-60, 60


|cFFFFFFFF - If the zone parameter is provided, it will only invite guild mates in that zone. If zone is left blank, it will only invite guild mates in your current zone. If zone is not provided, it will invite guild mates in any zone.

|cFF00FF00 > Examples: z-"Molten Core", z-""


|cFFFFFFFF - If the rank parameter is provided, it will only invite guild mates of that rank and higher. If rank is not provided, it will invite players of any rank.

|cFF00FF00 > Examples: r-"Raider", r-"Member"


Whisper Commands


|cFFFF0000The following commands may be whispered to players running Øne Raid.


|cFFFF0000* Requires the user to provide the correct password that the recipient has set. This command will fail if the recipient has never set a password using the
/orpassword command.


follow

|cFFFFFFFFMakes the user autofollow you.


join

|cFFFFFFFFMakes the user accept a group invitation.


decline

|cFFFFFFFFMakes the user cancel a group invitation.


disband

|cFFFFFFFFMakes the user leave a group.


promote password *

|cFFFFFFFFMakes the user promote you to group leader, if the user is the current leader.


trade password *

|cFFFFFFFFMakes the user accept a trade.


rez

|cFFFFFFFFMakes the user accept a rez request.


summon

|cFFFFFFFFMakes the user accept a summon request.
