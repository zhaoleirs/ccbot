This is a bit outdated but most of the information here remains the same, and for that reason there's no point in just completely deleting this file.
All In One Inventory
/aioi
Allows you to replace your normal bags with one big bag, with adjustable width and height.
Atlas
/atlas
Allows you to browse through a collect of instance maps. This is toggle-able via the Features Frame and UUI Menu.
Available Only
N/A
This hides unavailable spells/skills by default when visiting a trainer. You can see unavailable spells/skills by checking the "unavailable" box in the trainer filter as normal.
You can disable it at the character selection screen.
AvgXP
/avgxp
Calculates Average Exp Gain and puts it in a small frame with other various xp stats.
BankStatement
/bs
View the contents of your bank, anywhere in the world, including quantity of each item. You can also view the inventories of your alternate characters on different realms. Toggleable via the UUI Menu as well as the slash command.
BetterKeyBinding
N/A
Provides a better key binding frame for the binding of various UI features. Toggleable via the UUI Features Frame and the UUI Menu.
BuffTimers
N/A
Adds mini buff timers under the top right buff icons. Disable via the character selection screen.
ChatScroll
N/A
Makes chat windows scrollable with the mouse wheel.
ChatTimestamp
N/A
Timestamps chat messages as they come in. Disable via the character selection screen.
Clock
/clock
Adds a clock window with time-based tooltip information.
CombatStats
/cs
Shows key combat statistics about every attack/action you perform.
CooldownCount
/cc
Allows you to see time until actions have cooled down on the actual buttons.
CooldownCountFlexBar
N/A
Allows CooldownCount to work with Flexbar.
DefendYourself
/dy
Attacks your current target as long as you are in combat, along with various other automated combat actions.
DynamicData
N/A
Library required for All In One Inventory.
DynamicDataItem
N/A
Another Library required for All In One Inventory.
EquipCompare
N/A
Adds a tooltip to compare items easily with ones you have equipped. Disable at the character selection screen.
EquipManager
/equipmanager
Adds a handy manager to switching between equipment sets. Toggleable via the UUI Menu.
Fetch
N/A
Addon used to grab a characters real name, required for DefendYourself.
FlexBar
/flexbar
Provides access to 96 incredibly versatile buttons. Can be used for all of your button needs.
Framerate
N/A
Displays your framerate in a moveable frame.
FriendsFacts
N/A
Remember friends level, class and location after they've logged off. Disable at the character selection screen.
Gatherer
/gatherer
Displays the location of the mineral viens / herbs / chests on your minimap and regular map.
GoodInspect
N/A
Increases inspect range and adds guild info to the inspect window.
GoodTip
/goodtip
Alters tooltip appearance with various options, such as tooltip coloring and guild tag addition. The main features of GoodTip can be toggled via the UUI Menu.
Gypsy_BuffBar
N/A
Enhanced player effects display.
Gypsy_General
N/A
General interface customizations.
Gypsy_Hotbar
N/A
Removes artwork from the default action bar and adds an additional row.
Gypsy_Shell
N/A
Add-on configuration system for the other Gypsy addons.
Gypsy_UnitBars
N/A
Player, target, and party modifications. Allows them to be dragged anywhere on the screen, among various other options. Options are configurable via the "G" button on the minimap.
InfoBar
N/A
The InfoBar is a bar for presenting at-a-glance information through an easily customized plugin architecture. Individual plugins can display information on the bar or its summary tooltip and are configured through the InfoBar's interface. Disable at the character selection screen.
ItemBuff
/itembuff
Displays icons and timer text for buffs cast on your items in a small movable window.
Karma
/karma window
Rating system for players in your faction. Accessable via the UUI Features frame and the UUI Menu.
LootLink
/lootlink
An in-game browseable, searchable item database, with loot link coloring based on rarity. Toggleable via the UUI Features frame.
MapCoords
N/A
Shows cursor & player coords on map.
MapNotes
/mapnote
Adds a note system to the Worldmap.
MobileFrames
See Below
Makes selected frames mobile by dragging their top bar. Slash Commands: "/mflist" - lists currently mobile frames. "/mfreset [Frame]" - resets the position of that mobile frame. No frame resets all. "/mfenable [Frame]" - enables that mobile frame. No frame enables all. "/mfdisable [Frame]" - disables that mobile frame. No frame disables all.
MoneyDisplay
N/A
Small movable window displaying how much money you have.
NoteIt
/ni
Allows you to create notes about Players, NPCs and Items. Toggleable via the UUI Features frame and the UUI Menu.
QuickLoot
/quickloot
Automatically positions the most relevant part of the loot window under your cursor.
Reputation
/reputation
Shows detailed reputation numbers in the reputation frame.
SimpleMail
N/A
Preserve recipient's name and autofill subject line in mail forms.
SocialSendMessage
N/A
Adds "Send Message" and "Send Page" buttons and Player notes to the Guild/Who/Friends frame.
TackleBox
/tacklebox
Makes the fishermans life much easier. You can auto cast by simply right-clicking with a pole equipped, and allows you to easily switch between your pole and weapon.
UUICore
N/A
The core coding for the UUI features frame and the UUI Menu.
WeaponButtons
/weaponbutton
Gives you a little window that contains your main hand weapon and your off hand weapon for easy viewing and switching.
WeaponQuickSwap
N/A
Function for quickly swapping between 2 weapon sets. This is for use in macros. Example:
WeaponSwap("*","*","Fist of Stone","Gizlock's Hypertech Buckler");
ZoomMap
N/A
Makes the minimap much bigger and easier to view. Toggleable via the UUI Features frame.
